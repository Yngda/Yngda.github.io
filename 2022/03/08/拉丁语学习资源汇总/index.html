如题。会不断更新。

在更：

  • latintutorial, 官网（其实也没啥内容）

  • Latinum - YouTube

  • polýMATHY - YouTube

已弃更：

  • Dwane Thomas - YouTube

其它：

  • 自己整理的一些视频

网站：

付费：

  • Learn Latin - Ancient Language Institute

  • Latinum

免费：

  • PHONETICA LATINÆ - How to pronounce Latin，网页附带了拉丁语字母的发音。由于拉丁语特殊的发音规则，有这一个网页就能掌握拉丁语的全部发音了: )

  • 拉丁语发音 - 维基百科，自由的百科全书

  • The Latin Library — 提供了可在线阅读的丰富的拉丁语文档

  • Learn medieval Latin - Latin — 来自于英国国家档案馆，提供了比较浅显，容易理解的拉丁语教程，非常适合新手

书籍：

  • 《拉丁语语法新编》中图网 - 内容全面，讲解详尽
  • 韦氏拉丁语 - Wheelock’s Latin - Wikipedia，官网
  • 剑桥拉丁语系列 - Cambridge Latin Course - Wikipedia，B 站有视频教程

词典

  • Victionarium - 维基多语词典
  • translatus‎ (Latin): meaning, origin, translation - WordSense Dictionary - 也可以查询英语
